## 📝 概要

Figmaで作成したデザインカンプをもとに、HTML/CSSでコーディングしたポートフォリオサイトです。

---

## 🛠 使用技術

・ HTML
・CSS
・JavaScript
・レスポンシブ対応（スマホ対応）
※Wordpress内へオリジナルテーマとして組み込み済み
※カスタム投稿機能を追加し、一部の写真などを簡単に差し替えられるようにしています。

※Wordpressへの接続環境内でしか確認ができません。
